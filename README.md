# React Webpack Starter

A starter project with webpack 4, babel 7, jest, enzyme, eslint and prettier

### Setup:

- `$ yarn`

### Develop:

- `$ yarn start`
- Then open http://localhost:8080/ to see your app.

### Test:

- `$ yarn test`

### Build:

- `$ yarn build`
- Your app will be on the dist folder.

### Lint:

- `$ yarn lint`

### Format:

- `$ yarn format`
